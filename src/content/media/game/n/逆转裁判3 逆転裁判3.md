仔细推敲下其实3代的整体质量确实比不上一代，3-2、3-3比较无聊，3-1、3-4体量又偏小，但是3-5实在是过于精彩。不仅仅是人物塑造和罪案剧情抓人，更是在超长流程中展现了高超的节奏把控和关卡设计水准。几个心锁和剧情相互配合，对于游戏流程的分割恰到好处，最终的游戏体验可谓张弛有度又无比流畅，不单单是”文字”、”剧本”的成功，更是游戏设计的胜利。逆转3光凭借最后一个案子，在我心中的地位也要比幽灵诡计高出一档。
